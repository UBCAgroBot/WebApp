# Setup

- install pyenv (https://github.com/pyenv/pyenv?tab%253Dreadme-ov-file#installation)

```bash
brew update
brew install pyenv
```

- install python 3.11

```bash
pyenv install 3.11
```

- install poetry (https://python-poetry.org/docs/)

```bash
curl -sSL https://install.python-poetry.org | python3 -
```

- start shell

```bash
poetry shell
```

- run code

```bash
python camera.py
```
